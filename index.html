TOPIK SKRIPSI

Untuk topik Kerja Praktek saya yaitu ada pada bidang pengembangan website terfokus kepada frontend.

ANALISIS SWOT

BERIKUT ANALISIS SWOT YANG BISA SAYA BERIKAN

Strenght (kekuatan)

User Interface modern & responsif → dengan fokus pada frontend, website bisa tampil menarik, nyaman digunakan di desktop maupun mobile. Penggunaan framework (Bootstrap, Tailwind, React, dll.) → mempercepat proses pengembangan dengan komponen siap pakai. Kemudahan akses → tidak perlu instalasi khusus, cukup melalui browser. Customisasi visual → fleksibilitas dalam mendesain tampilan sesuai kebutuhan pengguna atau tema tertentu. Fokus pada pengalaman pengguna (UX) → lebih mudah menarik perhatian audiens karena tampilan interaktif.

Weaknesses (Kelemahan)

Kurang kuat di sisi backend → jika hanya fokus frontend, mungkin fungsi pengolahan data, keamanan, dan integrasi API jadi terbatas. Ketergantungan pada framework/CSS library → desain bisa terlihat generik jika tidak dikustomisasi. Tidak semua pengguna paham teknologi baru → jika desain terlalu modern, bisa membingungkan pengguna awam. Testing terbatas → pengujian lebih ke tampilan, belum mencakup skala besar (misalnya load balancing atau keamanan).

Opportunities (Peluang)

Kebutuhan informasi berbasis web semakin tinggi → banyak masyarakat lebih suka mengakses informasi lewat website responsif. Tren Mobile First → website dengan frontend yang responsif bisa menjangkau lebih banyak pengguna HP. Integrasi dengan teknologi baru → bisa dikembangkan ke arah PWA (Progressive Web App) atau integrasi dengan API kesehatan. Potensi jadi portofolio → hasil pengembangan bisa jadi bukti keterampilan frontend development untuk karier setelah lulus. Dukungan framework modern → banyak komunitas dan dokumentasi yang mempermudah pengembangan dan pemeliharaan.

Threats (Ancaman)

Persaingan ketat → sudah banyak website serupa dengan tampilan yang lebih profesional. Perubahan tren desain cepat → frontend yang dianggap modern sekarang bisa ketinggalan dalam 1–2 tahun. Ketergantungan pada koneksi internet → tanpa backend lokal atau mode offline, website sulit digunakan di daerah dengan jaringan terbatas. Keamanan terbatas → jika frontend tidak dipadukan dengan backend yang aman, rawan serangan (misalnya XSS). Keterbatasan nilai akademis → jika hanya frontend statis, penelitian bisa dianggap kurang “ilmiah” kecuali ada pendekatan khusus (misal: uji usability, eksperimen desain UI/UX).

Analisis Porter

berikut analis porter yang saya berikan

Threat of New Entrants (Ancaman Pendatang Baru)

Tinggi → membuat website frontend relatif mudah, tools seperti Bootstrap, Tailwind, atau template sudah banyak tersedia gratis. Mahasiswa atau developer pemula bisa cepat membuat website serupa. Dampak ke skripsi: Supaya lebih kuat secara akademis, kamu perlu menambahkan keunikan (UI/UX testing, desain interaktif, fokus pada edukasi kesehatan tertentu).

Bargaining Power of Suppliers (Kekuatan Tawar Pemasok)

Dalam konteks frontend website, pemasok = framework, library, atau platform hosting. Sedang → banyak pilihan open-source (Bootstrap, React, Tailwind, GitHub Pages, Netlify) yang gratis dan mudah digunakan. Risiko: kalau terlalu bergantung pada satu framework, bisa membatasi fleksibilitas desain.

Bargaining Power of Buyers (Kekuatan Tawar Pengguna/Audiens)

Tinggi → pengguna sekarang punya standar tinggi terhadap tampilan (harus cepat, responsif, dan enak dilihat di HP). Jika website tidak menarik atau lambat, pengguna langsung meninggalkan. Implikasi skripsi: penting untuk menguji usability, kecepatan, dan pengalaman pengguna (UX testing) sebagai bagian dari penelitian.

Threat of Substitute Products (Ancaman Produk Pengganti)

Tinggi → banyak alternatif lain seperti aplikasi mobile (Android/iOS), platform blog (WordPress, Medium), atau media sosial (Instagram, TikTok) yang bisa jadi sumber informasi kesehatan. Website frontend bisa kalah saing jika tidak menawarkan fitur atau tampilan yang lebih baik. Supaya relevan, website harus punya nilai tambah → misalnya tampilan edukatif interaktif, navigasi yang ramah pengguna, atau fokus pada topik spesifik (contoh: penyakit cacingan).

Industry Rivalry (Persaingan dalam Industri)

Sangat Tinggi → banyak website kesehatan, portal berita, dan aplikasi medis dengan desain modern. Tantangan: membedakan website skripsi dari website besar seperti Halodoc, Alodokter, KlikDokter. Peluang: fokus pada niche market (misalnya edukasi kesehatan anak, penyakit tertentu, atau konten lokal berbasis bahasa daerah) untuk mengurangi persaingan langsung.
